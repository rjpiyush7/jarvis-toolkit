🧠 What is XSS?

XSS (Cross-Site Scripting) is a security problem where a website shows what a user types without checking it properly.


⚠ Example 1 (Safe case)

If you type: Hello World
It will show: Hello World


⚠ Example 2 (Unsafe case)

If you type: <b>Hello</b>
It will display: Hello (bold text)


⚠ Example 3 (Style injection case)

If you type: <p style="color:red;">Red text</p>
It will display: Red text


⚠ Example 4 (Database Injection concept)

If a website directly uses user input in a database query without checking it properly.

Example input: ' OR '1'='1

This can trick the system into returning all data instead of checking valid login details.

❗ This happens because the browser treats input as HTML instead of plain text.


🔐 Always remember: Never trust user input directly — always sanitize it before displaying.
