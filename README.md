# Jarvis-Toolkit
A sleek, Flask-powered Cyber Awareness Toolkit. Features password entropy analysis, URL threat detection, and interactive security simulations for educational purposes.


JARVIS — Cyber Awareness Toolkit (v4.2.1)
JARVIS (Joint Artificial Intelligence Response & Vulnerability Inspection System) is an advanced, web-based educational framework designed to demonstrate common cyber threat vectors and provide analysis tools for modern security awareness.

Built with a high-fidelity "Cyber-Neon" interface, it bridges the gap between complex security concepts and user-friendly interaction.

🛠 Core Security Modules

MOD — 01: Password Checker – Multi-factor entropy scoring to evaluate credential strength.

MOD — 02: URL Analyzer – Scans links for phishing patterns, malicious redirects, and WHOIS domain age.

MOD — 03: Keylogger Simulation – Educational demonstration of how keystroke capture mechanics function.

MOD — 04: Brute Force Simulator – Visualizes credential exhaustion attacks using dictionary vectors.

MOD — 05: Phishing Demo – Interactive builder to help users identify social engineering tactics.

MOD — 06: XSS Laboratory – Real-time Cross-Site Scripting injection and defense playground.

💻 Tech Stack
Backend: Python (Flask)

API Integrations: VirusTotal API, WHOIS, DNS Lookup

Frontend: HTML5, CSS3 (Modern Grid/Flexbox), JavaScript (ES6+)

Deployment: Render / Vercel
